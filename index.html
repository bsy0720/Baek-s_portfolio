Work Plan

  • 💻 Java Script(ES6)

    ie가 지원이 종료되며 크로스브라우징의 의미가 없어지고 프레임워크에 사용이 많아지면서 ES6문법은 필수가 되었습니다. 그렇기 때문에 라이브러리 없이도 자유자재로 쓸 수 있도록 공부중입니다.

  • 💻 GIT

    IT직종은 협업을 많이 하기 때문에 소통이 중요한 직종입니다. 그렇기 때문에 협업을 위해 필요한 GIT을 좀 더 전문적으로 공부하려고 합니다.

  • 💻 REACT

    인터랙티브한 웹사이트를 만들 때에는 정적 웹사이트와는 다르게 때에 따라 달라지는 화면을 보여줘야 하기에 이런 상황에 딱 적합한 프레임워크인 리액트를 이해하고 사용할 수 있게 공부하겠습니다.

  • 💻 Node.js

    프론트엔드와 백엔드 개발자와의 원할한 소통을 위해 node js도 공부할 예정입니다.

  • 💻 TYPE SCRIPT

    자바스크립트 이외에도 많이 사용되어지고 있는 것이 TYPE SCRIPT라고 생각합니다. 그렇기 때문에 TYPE SCRIPT도 자유자재로 구현할 수 있도록 공부하겠습니다.

  • 💻 NEXT.js

    React의 서버사이드렌더링을 쉽게 구현할 수 있게 도와주는 프레임워크입니다. React를 배우며 함께 공부해 나가겠습니다.
